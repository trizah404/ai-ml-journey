## Projects Built

### Legal Document Q&A — RAG Pipeline
Built a RAG system from scratch without LangChain. PDF ingestion, 
semantic chunking, cosine similarity retrieval, Groq LLM with 
cited answers.

- [Live Demo](https://trizah404.github.io/legal-doc-qa)
- [GitHub](https://github.com/trizah404/legal-doc-qa)
- Stack: Python · FastAPI · sentence-transformers · Groq · HuggingFace Spaces

### Nepali Sentiment Analysis
One of the few deployed NLP tools built specifically for Nepali 
language. Trained on 35,789 samples. Identified class imbalance 
and improved neutral recall from 18% to 53%.

- [Live Demo](https://trizah404.github.io/nepali-sentiment-analysis)
- [GitHub](https://github.com/trizah404/nepali-sentiment-analysis)
- Stack: Python · sentence-transformers · scikit-learn · FastAPI · HuggingFace Spaces
